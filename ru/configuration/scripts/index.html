Раздел Scripts

В секции scripts.sources описываются источники скриптов. В данный момент поддерживается только тип folder

  • folder
scripts:
  sources:
    folder:
      - name: scripts
        path: /opt/scripts
        mask: '*.lua'
        update_interval: 5s
